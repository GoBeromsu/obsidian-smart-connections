# Open Smart Connections `v1.0.0`

## Initial Release

This is the first release of Open Smart Connections, a community fork of Smart Connections v3.

### What's Included

- **Semantic Connections**: Surface semantically related notes using local embeddings
- **Lookup View**: Semantic search across your vault
- **Smart Chat**: AI-powered chat with your notes
- **Local Embeddings**: Privacy-focused embedding generation

### Changes from Upstream

- All dependencies merged into `lib/` for standalone repository
- Removed promotional content and external links
- Fixed obsidian-smart-env config merge for proper settings display

### Bug Fixes

- Fixed settings screen missing "Smart Environment" section and model settings
- Merged `obsidian-smart-env/smart_env.config.js` to include `env_settings`, `collection_settings` components

## Technical Notes

This fork is based on Smart Connections v3.0.80 and includes all jsbrains dependencies locally.
