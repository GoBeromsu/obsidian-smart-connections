# Open Smart Connections `v1.0.4`

## New Features

- **Added Upstage embedding API support**: You can now use Upstage's Solar embedding models directly from the settings
  - Select "Upstage (API)" from the embedding model platform dropdown
  - Available models: `embedding-query` (optimized for queries) and `embedding-passage` (optimized for passages)
  - 4,096 dimensions, 4,000 token limit
  - Get your API key from [console.upstage.ai](https://console.upstage.ai)

## Technical Changes

- `lib/obsidian-smart-env/default.config.js`: Added `SmartEmbedUpstageAdapter` import and registration to `smart_embed_model.adapters`
